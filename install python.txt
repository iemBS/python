Install Python
Note
  - Guidance from https://www.python.org/downloads/
Preconditions:
  - Downloading Python 3
  - Using 64 bit OS
  - zip files download faster
Main Success Scenario
  1. Go to website for latest Windows version
       https://www.python.org/downloads/windows/
  2. Choose "Windows x86-64 executable installer" file
  3. download file
  4. run python executable
  5. choose default location to install to
  6. Install complete
